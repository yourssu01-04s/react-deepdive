## 스터디 진행 방식

1. 각 챕터(장)를 읽고 정리합니다. 진행자는 매주 돌아가면서 진행합니다!
2. 책 읽으면서 궁금한 점, 의문점, 토론할 주제를 하나 이상 생각해 옵니다.
3. 정리한 내용을 간단하게 발표하고 내용에 대해 토론하는 시간을 가집니다.
